### 关于drupalyun

* 基于Drupal7.x
* 专注于企业开源平台

### 安装

#### 系统需求

* Drush & Drush make
* Linux (现在只基于Linux,随后会支持windows, BSD)
* Mysql + Nginx/Apache + PHP5.3+

#### 安装步骤

* 创建数据库

* 执行build.sh [workspace] [domain_name]
    > 注意: workspace 是装载Drupal核心的目录，默认是html;
    > domain_name 是你的域名名称，默认是 drupalyun.com

* 修改install.sh 脚本参数
    > 注意: 重要的参数是 db_url,请匹配你的服务器配置.

* 完成
